## SEB - AVERAGE OF THREE NUMBERS
---
### AIM  
To write a Python program to define a function that accepts 3 values and returns their average.

---
### ALGORITHM
1. Begin the program.  
2. Define a function `result(a, b, c)` with the following steps:  
   - Calculate the average using the formula `(a + b + c) / 3`.  
   - Print the average in the format `"average is <value>"`.  
3. Read three integers `a`, `b`, and `c` from the user using `input()`.  
4. Call the function `result(a, b, c)` by passing the three values.  
5. Terminate the program.

---
### PROGRAM
```python
# Reg.No- 212222060100
# Name- Jothivanan T
def result(a, b, c):
    average = (a + b + c) / 3
    print("average is", average)

a = int(input())
b = int(input())
c = int(input())
result(a, b, c)
```

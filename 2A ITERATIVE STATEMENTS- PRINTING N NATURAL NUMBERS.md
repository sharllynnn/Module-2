## ITERATIVE STATEMENTS – PRINTING NUMBERS FROM M TO N
### Aim
To create a Python program for printing numbers from M to N (including M and N values) using a `for` loop.

---
### Algorithm
1. Begin the program.
2. Use `input()` to read the value of `a` (starting number) from the user.
3. Use `input()` to read the value of `b` (ending number) from the user.
4. Convert both inputs to integers.
5. Use a `for` loop to iterate from `a` to `b` (inclusive).
6. In each iteration, print the current value of `i`.
7. Terminate the program.

---
### 🧾 Program
```python
# Reg.No- 212222060100
# Name- Jothivanan T
a = int(input())
b = int(input())
for i in range(a, b + 1):
    print(i)
```
